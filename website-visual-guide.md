# 个人网站完整交互体验说明
## Yiheng Liu · AI PM Portfolio

> 本文档以**用户旅程视角**完整描述从打开网站到浏览结束的全部交互过程。
> 不用"屏幕 1-8"的抽象说法，而是用"你会看到什么、能做什么"的方式讲清楚。

---

## 网站类型说明

**这是一个单页长滚动网站 (One Page Website)**，就像一张很长的海报，你从上往下滚动鼠标就能看完全部内容。

**不需要点击导航跳转页面**，所有内容都在一个页面上，按顺序排列。

**唯一的"弹出内容"**：点击三个项目（Jarvis/My Heart/ARF）时，会从左侧滑出一个抽屉展示详细信息。

---

## 视觉基调（整个网站的配色和字体）

**配色**:
- 底色: 温暖米白 `#F5F3EE`（像建筑图纸的纸张颜色，不刺眼）
- 主文字: 墨黑 `#1C1C1C`
- 点缀色: 琥珀橙 `#D97706`（只在按钮、链接、数字下划线等小地方用）
- 网格: 极淡灰 `#E5E2DC`（背景有一层几乎看不见的细网格）

**字体**:
- 英文标题: Monument Grotesk（或 Sentient）— 有性格但克制
- 中文标题: 思源黑体 Bold
- 技术数字/协议: JetBrains Mono（等宽字体，像代码编辑器）
- 正文: Inter（英文）/ 思源黑体 Regular（中文）

## 关于部署的完整说明

**推荐方案：Vercel + GitHub + 自定义域名**

**步骤**:
1. 代码上传到 GitHub 仓库
2. 在 Vercel 注册账号（免费）
3. 点"Import Project"，选择你的 GitHub 仓库
4. Vercel 自动识别 Next.js 项目并部署
5. 部署成功后，Vercel 给你一个默认域名：`你的名字.vercel.app`
6. 在 Vercel 设置里添加你自己的域名（如 `yihengliu.com`）
7. Vercel 自动配置 HTTPS 和 DNS
8. 以后每次你 push 代码到 GitHub，Vercel 自动重新部署

**费用**：完全免费（Vercel 免费额度对个人网站绰绰有余）
